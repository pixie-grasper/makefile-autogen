What is it?
================

Generator! You will get a Makefile from your Source Folders.

How to use
==========

1. Put autogen.sh into your project directory.
2. Run it.
`$ TARGET=foo ./autogen.sh`
3. You will get the Makefile.

Supporting Language
===================

* C (*.c)
* C++ (*.cc, *.cpp)
